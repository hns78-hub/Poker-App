# ♠️ Poker Ledger System

A premium, glassmorphism-themed web application designed for poker groups to track session results, visualize performance trends, and manage group standings.

![Poker Ledger](https://images.unsplash.com/photo-1541278107931-e006523116d9?q=80&w=1200&h=630&fit=crop)

## ✨ Features

- **Intuitive Parser**: Simply paste your raw session notes, and the system automatically calculates buy-ins, chip values, and net returns.
- **Zero-Sum Validation**: Automatic consistency check ensures the pot and the payouts match perfectly.
- **Insights Dashboard**:
  - **Winnings Over Time**: Interactive Chart.js line graph showing cumulative earnings for all players.
  - **Net Performance**: Visual comparison of all-time profit and loss.
  - **Game Frequency**: Track who is playing the most sessions.
- **Global Leaderboard**: Comprehensive rankings based on total net gain, capital invested, and games played.
- **Admin Panel**: Secure access to edit session data, adjust dates, or delete records.
- **Mobile Optimized**: Fully responsive UI designed to work perfectly on smartphones and tablets.
- **Safe Storage**: Uses browser `localStorage` for privacy and offline-first persistence.

## 🛠️ Technology Stack

- **Core**: Vanilla HTML5, CSS3, JavaScript (ES6)
- **Design**: Premium Glassmorphism UI with CSS custom properties
- **Visualization**: [Chart.js](https://www.chartjs.org/)
- **Icons**: [Lucide](https://lucide.dev/)
- **Deployment**: Optimized for [Vercel](https://vercel.com/)

## 🚀 How to Use

1. **Log a Session**: Go to "New Session" and paste your raw results (e.g., `Name +Buyins (Chips)`).
2. **Review & Save**: Check the consistency status and save to the ledger.
3. **Analyze**: Visit the "Dashboard" or "Leaderboard" to see group progression.
